\chapter*{پیش‌گفتار}
\addcontentsline{toc}{chapter}{پیش‌گفتار}

این یادداشت‌ها بر اساس مجموعه ویدیویی معروف «ذات جبر خطی» \lr{(Essence of Linear Algebra)} اثر \lr{Grant Sanderson} از کانال \lr{3Blue1Brown} تهیه شده است. هدف اصلی این مجموعه، ارائه درکی شهودی و هندسی از مفاهیم جبر خطی است.

\section*{ویژگی‌های این یادداشت‌ها}
\begin{itemize}
    \item \textbf{شهود هندسی:} تأکید بر درک تصویری مفاهیم، نه فقط محاسبات
    \item \textbf{تعاریف دقیق:} ارائه فرمول‌های ریاضی استاندارد
    \item \textbf{مثال‌های کاربردی:} کاربرد در فیزیک، علوم کامپیوتر و مهندسی
    \item \textbf{تمرین‌ها:} مسائل متنوع برای تثبیت یادگیری
\end{itemize}

\section*{پیش‌نیازها}
\begin{itemize}
    \item آشنایی با مفاهیم پایه ریاضیات (جبر دبیرستانی)
    \item آشنایی با دستگاه مختصات دکارتی
    \item علاقه به درک عمیق ریاضیات!
\end{itemize}

\section*{نحوه استفاده}
هر درس شامل بخش‌های زیر است:
\begin{description}
    \item[تعاریف:] مفاهیم کلیدی با نماد‌گذاری ریاضی
    \item[شهود هندسی:] توضیحات تصویری (در کادرهای آبی)
    \item[کاربرد عملی:] مثال‌های واقعی (در کادرهای سبز)
    \item[تمرین‌ها:] مسائل برای تمرین
\end{description}

\begin{flushright}
با آرزوی موفقیت در یادگیری جبر خطی
\end{flushright}
